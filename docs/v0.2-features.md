## ğŸ†• New Features in v0.2

### ğŸ” AI Auto-Redact
Automatically detect and redact sensitive information in screenshots.

- **Auto-detection**: Emails, phone numbers, API keys, passwords, credit cards, IP addresses
- **3 redaction styles**: Blur, Pixelate, Black Box
- **Preview before applying**: Review redaction results before saving
- **Integrated with capture flow**: Optional auto-redact when capturing

### ğŸ’» CLI Tool
Use MyShot from the Terminal with full feature support.

```bash
myshot capture --area              # Capture selected area
myshot capture --fullscreen        # Capture entire screen
myshot capture --window            # Capture window
myshot capture --area --redact     # Capture + auto redact
myshot record --area --format gif  # Record area as GIF
myshot ocr ~/image.png             # Extract text from image
myshot annotate ~/image.png        # Open annotation editor
```

**Options**:
- `--output, -o`: Output file path
- `--copy, -c`: Copy to clipboard
- `--format`: mp4 or gif (for record)
- `--json`: JSON output format

### ğŸ“œ Scrolling Capture
Capture content longer than the screen (web pages, documents).

- **Area selection**: Draw to select capture region
- **Multi-frame capture**: Capture multiple frames while scrolling
- **Auto-stitching**: Automatically stitch frames into one long image
